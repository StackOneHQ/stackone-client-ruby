# TaskSchemasValue

The unified value for the type of the task. If the provider does not specify this type, the value will be set to UnmappedValue


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `ACTION`         | action           |
| `REVIEW`         | review           |
| `ACKNOWLEDGMENT` | acknowledgment   |
| `EDIT`           | edit             |
| `APPROVE`        | approve          |
| `UNMAPPED_VALUE` | unmapped_value   |
