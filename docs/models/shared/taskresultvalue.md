# TaskResultValue

The unified value for the status of the task. If the provider does not specify this status, the value will be set to UnmappedValue


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `OPEN`           | open             |
| `IN_PROGRESS`    | in_progress      |
| `BLOCKED`        | blocked          |
| `COMPLETED`      | completed        |
| `CANCELLED`      | cancelled        |
| `UNMAPPED_VALUE` | unmapped_value   |
