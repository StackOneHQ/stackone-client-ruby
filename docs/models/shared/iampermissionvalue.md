# IamPermissionValue

The type of the permission, e.g. read, read_write, delete, etc.


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `READ`           | read             |
| `READ_WRITE`     | read_write       |
| `APPROVE`        | approve          |
| `DELETE`         | delete           |
| `USE`            | use              |
| `EXPORT`         | export           |
| `UNMAPPED_VALUE` | unmapped_value   |
