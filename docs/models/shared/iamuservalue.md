# IamUserValue

The status of the user, e.g. whether the user is enabled, has been disabled (eg. by an admin), or is pending (ie: awaiting approval by the user or an admin).


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `ENABLED`        | enabled          |
| `DISABLED`       | disabled         |
| `PENDING`        | pending          |
| `UNMAPPED_VALUE` | unmapped_value   |
