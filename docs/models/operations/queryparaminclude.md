# QueryParamInclude

The include parameter allows you to include additional data in the response.

## Example Usage

```ruby
require "stackone_client"

value = QueryParamInclude::STEP_LOGS
```


## Values

| Name            | Value           |
| --------------- | --------------- |
| `STEP_LOGS`     | step_logs       |
| `ADVANCED_LOGS` | advanced_logs   |
